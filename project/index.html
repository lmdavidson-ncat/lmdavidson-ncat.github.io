What is this game about?

This game is designed to help you learn basic HTML by typing code snippets in a fun and engaging way. Our goal is to improve your understanding of HTML syntax through an interactive typing game.


How the Game Works!

Lilly B's HTML Typing Game will display HTML code snippets that you need to type correctly within a certain time limit. As you progress, you'll learn more about HTML syntax such as tags, attributes, and structure.
